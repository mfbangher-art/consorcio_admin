# Consorcio – Sistema de Administración

Una aplicación sencilla (Tkinter + SQLite + FPDF) para gestionar ingresos, egresos, propietarios, rubros y generar reportes/PDFs.

## Requisitos

- Python 3.9+
- pip

## Instalación

```bash
git clone https://github.com/tu_usuario/consorcio.git
cd consorcio
python -m venv venv
source venv/bin/activate          # Windows: venv\Scripts\activate
pip install -r requirements.txt
